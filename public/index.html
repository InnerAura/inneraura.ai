01 The Document IS the Interface

Markdown Converged.
Visuals Didn't.

AGENTS.md, SKILL.md, docs, web apps — Notion, Obsidian, every knowledge tool — the entire developer and AI ecosystem converged on Markdown. But agents still learn design from text descriptions — reading about visuals instead of inspecting them. SVG is the only format that renders inside Markdown, parses as source code, and carries machine-readable metadata. Not a description of visual knowledge. The visual knowledge itself.
